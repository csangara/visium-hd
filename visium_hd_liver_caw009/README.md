## VisiumHD Liver CAW009 analysis

The scripts here are mostly identical to those of the liver SCA002 dataset, although there is less attempts at preprocessing the data.

The `ex_visualize_genes` contains a simple LR analysis, in which I calculated a null distribution of the distance between random ligand-receptor pairs and tested if Gdf2-Acvrl1 and Bmp10-Acvrl1 were significantly closer in this CAW009 dataset compared to the null distribution (it was not). But, as I am not so sure about the way I went about this, I did not include it in the chapter.
